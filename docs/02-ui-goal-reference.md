# UI Goal Reference (Visual Spec Summary)

Target UI style:
- Futuristic dark theme
- Starfield / constellation background
- Glassmorphism panels
- Neon borders with subtle glow
- Typography: clean sans for UI + monospace for code previews
- Cards with colored accent by language (JS green, Python cyan/blue, Docker purple, Bash teal, SQL blue)

Layout overview:
- Sticky translucent Navbar + neon divider line
- Center hero title: ">> Snippet Vault"
- Search bar (wide input + button)
- Meta line: "Resultados: X snippets"
- Grid of snippet cards (3 columns desktop)
- Pagination centered
