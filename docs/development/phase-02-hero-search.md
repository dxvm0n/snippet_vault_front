# Phase 02 — Hero Title + SearchBar

## Goal
Implement the central header block and search bar to match reference.

## Components
1) HeroTitle
2) SearchBar (compound)

## HeroTitle spec
- Content: ">> Snippet Vault"
- Alignment: centered
- Color: neon mint/green (brand)
- Size: 2.25–3rem range depending on breakpoint
- Glow: subtle text-shadow or filter drop-shadow (soft)

## SearchBar spec
Layout:
- single row: wide input + right button
- input takes ~75–82% width
- button takes remaining width, same height

Input:
- glass background, dark translucent
- neon border
- placeholder: "Buscar snippets..."
- focus: border becomes brighter + subtle glow ring

Button:
- outline neon border
- icon + label "Buscar"
- hover: slightly brighter border + background tint
