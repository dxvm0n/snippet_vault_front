# Phase 04 — SnippetGrid

## Goal
Create a responsive grid matching reference spacing.

## Component
SnippetGrid

## Behavior
- Renders list of snippets
- Does not modify data; only presents it
- Uses SnippetCard for each item

## Layout spec
- Desktop: 3 columns
- Large screens: maintain max-width 1320px
- Tablet: 2 columns
- Mobile: 1 column

## Spacing
- Gap: 24–32px
- Cards aligned top; equal widths per column
